# Welcome to the Nuxt Design System

The **Nuxt Design System** is a cutting-edge framework built with the power of **Nuxt 3**, **Vue 3**, **UnoCSS**, and **Radix UI**. It combines efficiency, customization, and modern design principles, allowing you to create stunning web applications with ease. üöÄ

---

## Why Nuxt Design System?

The Nuxt Design System leverages:

- **[UnoCSS](https://github.com/unocss/unocss):** The instant on-demand Atomic CSS engine.
- **[Radix UI](https://www.radix-ui.com):** A library of accessible and customizable UI components.
- **[Radix Colors](https://www.radix-ui.com/colors):** Beautiful, accessible color palettes designed for modern UI.

These tools are seamlessly integrated into the Nuxt 3 ecosystem, offering unparalleled flexibility and simplicity. üé®

---

## Getting Started

To start using the Nuxt Design System, follow these steps:

1. **Install the required packages:**

```bash
pnpm install unocss @radix-ui/colors
```

2. **Configure UnoCSS in your `nuxt.config.ts`:**

```js
export default defineNuxtConfig({
  modules: ["@unocss/nuxt"],
  unocss: {
    presets: [
      require("@unocss/preset-uno")(),
      require("@unocss/preset-attributify")(),
    ],
  },
});
```

3. **Import and use Radix Colors:**

```css
@import "@radix-ui/colors";
```

### 1. Atomic CSS with UnoCSS
Easily style your components with an intuitive and performant Atomic CSS approach. UnoCSS provides a delightful developer experience with:

- **Preset integrations** to expand functionality.

Learn more about UnoCSS [here](https://github.com/unocss/unocss). ‚ú®

## Styling Guidelines

Our design system focuses on a **monochrome-minimalist** aesthetic. Below are key styling tips:

- **Background:** `--color-background`
- **Primary Accent (Teal):** `--teal-9`
- **Neutral Palette:** `--gray-5`, `--gray-9`, `--gray-12` for text
